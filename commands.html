anti nuke:

     ane,enableantinuke - enables anti nuke (server owner)

     and, antinukedisable - disables anti nuke

     anbot - allows any admin or mod to add bots

     ankick - allows any admin or mod to kick users

     anban - allows any admin or mod to ban users 

     danbot - disables any admin or mod to add bots (has to be turned off prior)

     dankick - disables any admin or mod to kick users (has to be turned off prior)

     danban - disables any admin or mod to ban users (has to be turned off prior)

     wl, whitelist <@user> - whitelists a user from antinuke

     uwl, unwhitelist <@user> - unwhitelists a user from antinuke

 

mod:

     ban <@user> - bans a user

     kick - kicks a user

     mute, timeout, tm, jail <time> - times out a user

     idban <userid> - bans a user via their ID

     tempban <@user> <time> - temporarily bans a user for a set amount of time

     censor <word/phrase> - automatically blacklists a word or phrase

     logs <channel-id> - puts user logs, message edits, automod, joins and leaves, etc in a certain channel 

     slogs - stops sending logs to a certain channel

     role <@user> - roles a user

     rrole <@user> - removes a role from a user

     nuke - deletes all messages in a channel

     purge <number> - deletes a certain amount of messages in a channel
